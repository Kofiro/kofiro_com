Only One


A unique logic-centered puzzle game.



Only One is a minimal logic puzzle game made for the smartphone, where each level has its own logic to be solved. In Only One, every level's solution has to do with something related to the number 1.

Reset a screen of multiple colors to be one color, join up multiple pieces to form one piece of an object or an image.

Puzzles like these and many more await puzzle game lovers in Only One. The number one will give you all the clues you need to succeed.

About The Developer

Kofi Austin also known online as Kofiro, is a one man independent game developer from Ghana, West Africa, and is currently residing in the heart of Accra, it's capital.

With over 10 mobile games published since 2018, Kofi is risking it all to fulfill his dreams of being able to make video games for a living.
