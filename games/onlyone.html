Only One


A unique logic-centered puzzle game.


A tricky logic puzzle game with a variety of levels

Let there be one!

Only One is a minimal logic puzzle game where each level has its own logic to be solved. Every level's solution has to do with something related to the number 1.

One color? One piece? or even the number one itself. Just let it be one.

And if in any case, you get stuck on a level after 45 seconds a hint icon would be available to provide you with some help.

Can you make it Only One?
